### 🧑🏻‍💻 开源项目 ([more](https://github.com/SWHL?tab=repositories))

- [PunctuationAcademy](https://github.com/SWHL/PunctuationAcademy): 中文标点符号练习应用：从内置规范文章或自定义 TXT/ 粘贴范文中随机选句，挖空标点后由用户在空位处填写，提交后给出答案对比与错误高亮。
- [markdown-auto-space](https://github.com/SWHL/markdown-auto-space): 在保存或格式化时，自动为 Markdown 中的中英文之间添加空格。
- [RapidVideOCR](https://github.com/SWHL/RapidVideOCR): 🎦 提取视频硬字幕并自动生成对应的 SRT 字幕文件。
- [AI-Competition-Collections](https://github.com/SWHL/AI-Competition-Collections): 收集整理各类人工智能竞赛（NLP, CV、语音、推荐系统等）的经验帖与技巧。
- [RapidVideOCRDesktop](https://github.com/SWHL/RapidVideOCRDesktop): 基于 PyQt 开发的 RapidVideOCR 桌面版本。
- [ChineseDocumentPDF](https://github.com/SWHL/ChineseDocumentPDF): 针对中文论文、财报、证券类 PDF 文档的版面与内容处理工具。
- [TrOCR-Formula-Rec](https://github.com/SWHL/TrOCR-Formula-Rec): 基于 TrOCR 和 UniMER-1M 数据集训练的数学公式识别模型。
- [TableRecognitionMetric](https://github.com/SWHL/TableRecognitionMetric): 表格结构识别任务的评估指标计算工具（支持 TEDS, S-TEDS 等）。
- [LGPMA_Infer](https://github.com/SWHL/LGPMA_Infer): 表格结构识别模型 LGPMA 的推理实现。
- [LLaMADemo](https://github.com/SWHL/LLaMADemo): LLaMA 7B 模型的本地运行演示。
- [ConvertLaTeXOCRToONNX](https://github.com/SWHL/ConvertLaTeXOCRToONNX): 将 LaTeX-OCR 模型转换为 ONNX 格式以提升推理效率。
- [PPLCNet2PyTorch](https://github.com/SWHL/PPLCNet2PyTorch): 将 PaddlePaddle 的 PP-LCNet 迁移为 PyTorch 实现并验证性能。
- [ConvertLOREToONNX](https://github.com/SWHL/ConvertLOREToONNX): 将 LORE 表格识别模型从 PyTorch 转换为 ONNX 格式。
- [FontsCollection](https://github.com/SWHL/FontsCollection): 汇总多种字体资源，包括手写体、韩文字体等，用于 OCR 合成数据。
- [ExtractOfficeContent](https://github.com/SWHL/ExtractOfficeContent): 从 Word, Excel, PPT 等 Office 文件中提取文本、表格和图像内容。
- [flux-sliders](https://github.com/SWHL/flux-sliders): 基于 sliders 项目的修改版，用于图像生成中的参数控制（如 Flux 模型）。
- [ProcessLaTeXFormulaTools](https://github.com/SWHL/ProcessLaTeXFormulaTools): 处理 LaTeX 公式的工具集，支持 KaTeX 渲染等。
- [TextRecMetric](https://github.com/SWHL/TextRecMetric): 计算文本识别算法（如 CRNN）的准确率指标。
- [PyTorchDDP](https://github.com/SWHL/PyTorchDDP): PyTorch 多机多卡分布式训练的可执行示例。
- [WeChat-AutoSendBless](https://github.com/SWHL/WeChat-AutoSendBless): 微信自动群发个性化祝福消息（支持 PC 和安卓）。
- [TextDetMetric](https://github.com/SWHL/TextDetMetric): 计算文本检测算法的 H-mean 等评估指标。
- [GetPyPiLatestVersion](https://github.com/SWHL/GetPyPiLatestVersion): 获取指定 Python 包在 PyPI 上的最新版本号。
- [label_image](https://github.com/SWHL/label_image): 图像标注工具 LabelImg 的复刻 / 学习版本，用于目标检测数据标注。
- [TinyGPT-V](https://github.com/SWHL/TinyGPT-V): 整理版 TinyGPT-V 学习仓库，配套博客使用。
- [MNN_Tutorial](https://github.com/SWHL/MNN_Tutorial): 阿里 MNN 推理引擎的快速上手示例合集。
- [Mobilenetv3_from_torchvision](https://github.com/SWHL/Mobilenetv3_from_torchvision): 从 torchvision 0.12 中独立提取的 MobileNetV3 模型代码。
- [pytorch-unet](https://github.com/SWHL/pytorch-unet): PyTorch 实现的 U-Net 分割网络。
- [SWHL.github.io](https://github.com/SWHL/SWHL.github.io): 个人主页或文档站点源码。
- [pyecharts-showDynamicTrajectory](https://github.com/SWHL/pyecharts-showDynamicTrajectory): 使用 pyecharts 绘制动态轨迹图的示例。
- [markdown-auto-space](https://github.com/SWHL/markdown-auto-space): 自动为中英文混排 Markdown 添加空格的 TypeScript 工具。
