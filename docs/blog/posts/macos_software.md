推荐自己在 macOS 上常用的小软件

<!-- more -->

以下这些软件，是我经过长期使用和多方比对后筛选出的平替优选，每一款都堪称同类中的佼佼者。更重要的是——它们全部免费！

| 软件名称             | 功能简介 |
|----------------------|----------|
| Amphetamine.app      | 防止 Mac 进入睡眠状态，可自定义触发条件（如特定应用运行时保持唤醒）。 |
| Maccy                | 轻量级剪贴板历史管理工具，支持快捷键调出历史记录并粘贴。 |
| Alfred               | 强大的生产力启动器，支持快速搜索、文件操作、自定义工作流（Workflows）等。 |
| 坚果云               | 国内知名的云同步网盘，支持文件自动同步、备份与跨设备共享。 |
| Pixpin               | 截图与录屏工具，支持滚动截图、标注、OCR 识别及 GIF 录制等功能。 |
| Input Source Pro     | 高级输入法切换工具，提供更流畅、智能的多语言键盘布局管理。 |
| Scroll Reverser      | 反转鼠标滚轮或触控板滚动方向，可单独设置垂直 / 水平方向的滚动行为。 |
| Rectangle            | 窗口管理工具，通过快捷键将窗口自动贴靠到屏幕指定区域（类似 Windows 的窗口贴边功能）。 |
| IINA             | 专为 macOS 设计的现代开源视频播放器，基于 mpv，支持高分辨率、字幕、快捷键、触控栏等，界面简洁且功能强大。 |
| 超级右键 Lite         | 增强 macOS 右键菜单功能，添加复制路径、新建文件、终端打开等实用选项。 |
| AppCleaner           | 应用卸载工具，可彻底删除应用程序及其关联的配置、缓存等残留文件。 |
| PDFgear              | 免费 PDF 工具套件，支持阅读、编辑、注释、转换、OCR 等功能。 |
