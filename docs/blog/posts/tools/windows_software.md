推荐自己在 Windows 上常用的小软件

<!-- more -->

以下这些软件，是我经过长期使用和多方比对后筛选出的平替优选，每一款都堪称同类中的佼佼者。更重要的是——它们全部免费！

| 软件名称             | 功能简介 |
|----------------------|----------|
| 坚果云               | 国内知名的云同步网盘，支持文件自动同步、备份与跨设备共享。|
| Pixpin               | 截图与录屏工具，支持滚动截图、标注、OCR 识别及 GIF 录制等功能。|
| Geek           | 应用卸载工具，可彻底删除应用程序及其关联的配置、缓存等残留文件。|
| PDFgear              | 免费 PDF 工具套件，支持阅读、编辑、注释、转换、OCR 等功能。|
| Everything              | 快速搜索文件 |
| 7-Zip              | 小巧免费简洁干净的解压软件 |
| Potplayer              | 无广，支持几乎所有常见的音视频格式，包括高清和 4K 视频，界面简洁清爽，运行流畅。|

## 小技巧

开启自带剪贴板功能：Win + V 即可。
