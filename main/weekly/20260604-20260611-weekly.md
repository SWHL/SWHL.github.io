## 摘录

### [卢涛南-高考随想](https://lutaonan.com/blog/gaokao/)

高考考得好固然好，但人生啊，不是一段解题的过程，而是一场终生的牌局。总结起来，人生无非四种玩法：一手好牌 / 打得稀烂；一手好牌 / 打出更大价值；一手烂牌 / 破罐破摔；一手烂牌 / 打成好牌。

每个人都有不同的打法，只要打得让自己感到快乐，就是好的打法。如果同时还能让身边的人也感到快乐，那可算是功德圆满了。

### [燃油车，真的卖不动了？](https://www.huxiu.com/article/4866097.html?f=rss)

这是因为，当消费者看到车价越来越便宜，就会极易形成负向循环，越是频繁降价，消费者就越觉得再等等还能更便宜，于是持币观望，推迟购车。

当降价从一种短期促销手段变成一种常态预期，它刺激需求的能力就迅速衰减，甚至开始反噬消费信心。

另外，经销商为了清理库存加大优惠，频繁降价又会损害老车主权益和二手车残值。残值走弱进一步降低消费者购买意愿，经销商也会减少展厅位置、销售人员和营销资源。

---

产品降价促销更像是饮鸩止渴。

### [普通人参考博物馆的意义？](https://daily.zhihu.com/story/9790259)

这些漫无目的的想法，似乎也没有任何实用价值。

但它们像一把锤子，一锤子下去，敲碎了你认为的，世界一直像是今天这样的幻觉。

你每天活在自己的那一亩三分地中，你的外卖、你的工位、你的通勤路线、你手机应用里那些算法推荐，早就在你的大脑中形成了一套极其高效但又极其狭隘的「世界模型」。

而现在，你站在这把匕首前面胡思乱想，实际上你的大脑已经开始疯狂的重新编译你对世界的理解。

这个过程心理学上叫做「默认模式网络（Default Mode Network）」激活，这个系统负责让你反思并且重新进行意义构建。

用我们更流行的词汇讲，就是博物馆会把你从你的认知茧房中强行拽出来。

---

回想自己去博物馆的经历，似乎也就是漫无目的逛，并没有去寻找什么意义。只想找那种偷得浮生半日闲的感觉。

### [尤雨溪：从文科生到框架创造者的可复制路径](https://polishme.app/council/c/558ff502-ea05-4ff3-a237-946bf74a964b)

Vue 的诞生不是“我要造一个框架”的野心先行，而是尤雨溪在 Google 工作期间，使用 Angular 1.x 处理复杂应用时，清晰感受到的具体不适：框架太重、学习曲线陡峭、在小型项目中引入成本过高。他做了一件看似简单却极关键的事——把 Angular 中自己最需要的响应式机制抽离开来，结合自己喜欢的 API 风格，用业余时间搓出一个最小可行版本。

这一策略的本质：先成为重度用户，再成为改造者。对于任何想从技术使用者转向技术创造者的人来说，最可靠的起点永远是你自己在真实工作中被反复折磨的那个痛点，而非市场报告或技术潮流。

---

这个比较赞同，我的很多项目也都是这么来的。我看到的好多优秀项目起源都是如此。造就伟大宫殿的人，可能自己也没想到过，他只是在解决自己的一些痛点而已。

普通人每天可以生出几千种想法，其中很多既没用，也不理性，甚至是邪恶的。这些精神垃圾通过许多不同的形式存在。比方说，耳虫现象，是说某段旋律，用我们的话讲，叫特别“魔性”，会挤进你脑袋里萦绕不散；还有就是一些负面的念头，比如说，“我不行”“我不干了”，等等

---

我就体会到了，的确，会时不时有段旋律萦绕脑海。
