# 随笔

> Canonical URL: https://swhl.github.io/main/blog/20260716_note/
> Markdown URL: https://swhl.github.io/main/blog/20260716_note.md

<!-- more -->

发现自己隔一段时间，脑海里就会回荡起之前看过的某个电影，内心就有一种再看一遍的想法。

最近再看的是：电影《乐园追放》。兴许是自己看了电影《攻壳特工队》的缘故。因为我发现两个电影有某些相似之处，两者都出现了人类电子化的设定。这又让我想到了电影《流浪地球》。这真是一个扣一个呀！

这给我的感觉就是：
大脑就是一个超级加工厂，将
之前看过的书籍和电影，都在脑海里汇总，融合。

这一点从电影《垫底辣妹》中也有提及。女主想要考取庆应大学，但是一些基本常识不够，补习班老师就和她说：“读书多了，自然就补上来了”。

我比较信奉的是：开卷有益。不管看电影，还是阅读，都是在滋润着自己，潜移默化地影响自己。

此刻脑海中回荡的电影是：《无敌破坏王 2》，大概是因为里面映射了现代社会。

不知看这篇文章的你，有啥想看的吗？
